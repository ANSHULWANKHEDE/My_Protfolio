🌐 Developer Portfolio — React + Vite + Tailwind CSS

This repository contains my personal developer portfolio built using React (Vite) and Tailwind CSS.
The site includes an interactive About section, Skills, Projects, Experience timeline, and a PDF resume download integrated via Vite public assets.
Designed with a clean dark theme, responsive layout, and modular React components.

🚀 Features

⚡ Built with Vite — super fast development environment

🎨 Dark theme UI — modern, sleek, and easy on the eyes

📱 Fully responsive — works smoothly on all devices

📄 Resume Download — PDF stored in public/ for direct download

🧩 Modular components — clean and maintainable React structure

🎯 Developer-focused design — good typography, spacing, and layout visibility

📦 Easy Deploy — supports GitHub Pages / Netlify / Vercel

🛠️ Tech Stack

React (Vite)

Tailwind CSS

JavaScript (ES6+)

HTML5 / CSS3

Node.js & npm
